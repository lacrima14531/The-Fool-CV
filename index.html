HELLO
I'M KUAN YU

「踏上設計的不歸路」
回憶起小時候，由於家庭因素，早早就開始了打工生活，這也讓我害怕起了未來，我究竟想要做什麼。
即使是在高中時仍舊很迷惘，但最終還是在志願表上選了自己比較喜歡的科系「設計」。
最終與母親抗爭一番後，如願了進入了設計系。

「大學時的迷惘」
大學期間真的見識了許多的人事物，學得越多，越覺得自己渺小，因此我不斷的想辦法增強自己的能力
，卻也在不知不覺遇到了瓶頸...也是在這個時候，我開始同步參加了不同設計領域的學程及講座等，除了
增加自己不同領域的認識外，也加強自己的邏輯觀念及解決問題的能力，渴望能夠將我學習到的各領域
能力融會貫通後嘗試出不一樣的結果，因此大學生活過得非常充實

「設計有需要讀到研究所？」
這句話是我至今仍很常聽見的詢問，畢竟一切的專業都需要有「實務經驗」，因此畢業後早點投入社會
是大多數人的首選，如果大學是「遇到問題，解決問題」那研究所就是「遇到問題，如何用有邏輯及效
率的方法解決問題」。

「邏輯思考能力」及「表達能力」
相較於大學時期的「跳躍式思維」，有邏輯性的將資料統整分析後更能夠說服別人，使自己的作品更有
內涵外也能針對問題給予合適的回應，因此我個人認為，邏輯思考及口條表達能力便是重點學習內容。

專業技能

AutoCAD

利用現場丈量之實際尺寸繪製現況圖面、主體圖面、室內配置圖面等各種所需圖面資料。

SketchUp

配合與客戶討論出的平面配置圖繪製3D示意模擬供客戶及上司討論使用，更加直白的讓人看見實際完工時的模樣，減少誤會。

Illustrator

利用向量繪圖方式繪製Logo或icon等圖形創作，同時又可進行設計排版，運用在各領域範圍，是平面設計師的常用軟體。

Photoshop

針對照片及圖像進行調色或合成後製，適合用來處理相片及圖象，不僅是設計師的必備技能，也是許多行銷人員常用的軟體。

Enscape

Enscape為目前相關科系最常使用的即時渲染軟體，能與Revit、SketchUp、Rhino、Archicad 和 Vectorworks等各種軟體配合使用。

Twinmotion

同為即時渲染軟體，但較為常用於動畫渲染，內建資源相當豐富，同時也支援FBX、SKY、C4D、OBJ等等各種檔案模式，經常與其他繪圖軟體合作使用。

HTML

HTML是一種網頁使用的語言，可以用來設計出網頁架構，也可以在網頁中加入所有HTML語言可支援的方式，例如表格、表單、圖片、文字、連結、程式等等。

CSS

CSS是是一種用來為結構化文件添加樣式（字型、間距和顏色等）的電腦語言，可以透過 CSS 自訂網頁字體樣式、顏色，網頁排版橫排、直排、動畫等，。

過往經歷

之前 - 2011

高中時原本就想讀設計類相關科系~卻陰錯陽差的讀了資訊科~但也因為這美麗的誤會幫我奠定了電腦相關知識的基礎。

2011 - 2015

學校的課程相當多元，除空間設計類外還有程式設計及測量，因此學習了許多跨領域的知識與專長，而在百般思考下決定同步進行網頁設計的專題及新一代設計展，因而開始進行魔鬼般的培訓。

2015 - 2018

碩士畢業於嘉南藥理大學應用空間資訊系研究所，於碩士期間精進了自己不足的部分，特別是邏輯能力及口條，當然，過程中難免經歷了許多的失敗，但也是那些失敗成就了現在的我。

2018 - 2019

兵役期間在台南市歸仁區公所內擔任社區營造替代役，在那期間度過了一段非常有意義的生活，並參加了許多的公益活動，除了設計規劃外還需親自跑廠商和下工地，從鋪磚拌水泥到種花鋪草皮，歷練了一年的時間成為一位刻苦耐勞的有為青年！

2020 - 2023

於知名連鎖通路擔任了店鋪規劃副主任，並與協力廠商及各部門執行了大大小小不同程度的專案內容等相關業務，最終，專案執行約為7件、展店數合計為59間，有效的增加品牌知名度與市占率。
